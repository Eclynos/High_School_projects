# Photocheap

### Photocheap is a remake of [Adobe Photoshop](https://www.adobe.com/fr/products/photoshop.html) (i think you got the pun).
### I utilized the [tkinter](https://docs.python.org/3/library/tkinter.html) library in Python to code software that enables users to effortlessly generate entertaining or impressive photo montages.
